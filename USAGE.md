# PID 밸브 뷰어 사용 가이드

## 🚀 실행 방법

```bash
cd C:\Users\USER\Downloads\pid-viewer
npm run dev
```

브라우저에서 http://localhost:3006 접속

## 📖 사용 방법

### 1. 밸브 검색하기
- 상단 검색창에 밸브 태그 입력
- 예시: `V-102`, `HV-6003`, `VF-5027`, `VG-5007`
- 자동완성 결과가 드롭다운으로 표시됩니다

### 2. 밸브 선택하기
- 검색 결과에서 원하는 밸브 클릭
- 오른쪽에 상세 정보 패널이 나타납니다
- 도면이 자동으로 로드됩니다

### 3. 도면 보기
밸브를 선택하면:
- ✅ PDF 도면이 중앙에 표시됩니다
- ✅ 선택된 밸브 위치가 하이라이트됩니다
- ✅ 도면을 확대/축소할 수 있습니다

### 4. 상세 정보 확인
오른쪽 패널에서:
- 밸브 상태 (정상 작동/점검중/작동중지)
- 기술 사양 (압력, 온도, 재질 등)
- 위치 정보
- 관련 문서 링크

## 🎮 컨트롤

### 검색창
- `Enter`: 검색 실행
- `Esc` 또는 `X` 버튼: 검색 초기화

### 도면 뷰어
- **확대**: 하단 컨트롤바의 `+` 버튼
- **축소**: 하단 컨트롤바의 `-` 버튼
- **원본 크기**: 하단 컨트롤바의 `□` 버튼
- **페이지 이동**: 좌우 화살표 버튼

### 상세 패널
- **닫기**: 우측 상단 `X` 버튼

## 📂 데이터 파일 위치

### 밸브 데이터
- 경로: `public/data/valve_data.json`
- 형식:
```json
{
  "위치명": [
    "밸브태그1",
    "밸브태그2"
  ]
}
```

### PDF 도면
- 경로: `public/pdfs/`
- 현재 등록된 도면:
  - `010-1M-PI-111-002  DH HE Condensate System-2(R2).pdf`

## 🔧 추가 도면 등록 방법

1. PDF 파일을 `public/pdfs/` 폴더에 복사
2. `components/DrawingViewer.tsx`에서 PDF URL 업데이트
3. 또는 밸브 데이터에 도면 경로 추가

## 💡 팁

- 검색은 밸브 태그와 위치 모두에서 작동합니다
- 부분 검색도 가능합니다 (예: "HV"로 검색하면 모든 HV 밸브 표시)
- 여러 밸브를 연속으로 검색하여 비교할 수 있습니다

## 🐛 문제 해결

### 도면이 로드되지 않는 경우
1. PDF 파일이 `public/pdfs/` 폴더에 있는지 확인
2. 브라우저 콘솔(F12)에서 에러 메시지 확인
3. 개발 서버 재시작: `Ctrl+C` 후 `npm run dev`

### 검색 결과가 없는 경우
1. `public/data/valve_data.json` 파일 확인
2. 밸브 태그 철자 확인
3. 대소문자는 구분하지 않습니다
